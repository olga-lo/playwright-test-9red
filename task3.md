# Задача
Посчитаем возможные исходы, где при броске монеты 3 раза, получим два орла. Обозначим орел О, решка Р
1) О О О
2) О О Р
3) О Р О
4) О Р Р
5) Р О О
6) Р О Р
7) Р Р О
8) Р Р Р

Отберем те исходы, где выпало два орла:
1) О О P
2) О P O
5) Р О О

Посчитаем вероятность по формуле: количество нужных исходов / общее количество исходов
Вероятность = 3 / 8 = 0,375

Вероятность того, что при подбрасывании монеты три раза выпадет ровно два орла равна 0,375
