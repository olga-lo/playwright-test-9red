# Тестовое задание
Проект содержит тест-кейсы для веб-приложения оплаты картой, тест автоматизации Playwright и ответ на задачу по теории вероятности.


### 1. Тест-кейсы для веб-приложения оплаты картой
- Покрывает: позитивные сценарии, негативные сценарии, проверку граничных значений, валидацию и интеграцию с платежным шлюзом.
- Формат тест-кейсов:
  - ID
  - Название
  - Предусловия
  - Шаги
  - Ожидаемый результат
- Файл с тест-кейсами: [task1.md](./task1.md)

### 2. Задача Playwright
- Написать тест на страницу [Playwright](https://playwright.dev/) и проверить, что заголовок соответствует ожидаемому значению.
- Тест запускается минимум на двух браузерах.
- Ожидаемый результат: успешное прохождение проверки заголовка.
- Файл с задачей: [task2.js](./tests/playwright-dev.spec.js)

### 3. Задача по теории вероятности
- Условие: Монету подбрасывают три раза. Найти вероятность того, что выпадет ровно два орла.
- Ожидаемый результат: решение и ответ
- Файл с задачей: [task3.md](./task3.md)

### Установка и запуск тестов

- скачать проект `git clone  https://github.com/olga-lo/playwright-test-9red.git`
- установаить зависимости `npm install`
- запустить тесты `npx playwrite test`
